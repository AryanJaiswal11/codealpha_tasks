#TASK 1. Student Grade Tracker

## Overview

A console-based Java application developed to manage and analyze student academic performance. The system allows users to store student marks, calculate average scores, identify highest and lowest marks, and generate summary reports. The project demonstrates the use of Object-Oriented Programming principles, collections, and data processing techniques to efficiently manage student records.

## Features

* Add and manage student records
* Calculate average scores
* Find highest and lowest marks
* Generate performance summaries
* Store and process data using Arrays and ArrayLists
* Console-based user interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collections Framework (ArrayList)

#TASK 2. Hotel Reservation System

## Overview

A console-based Hotel Reservation Management System built using Java, Spring Boot, Spring Data JPA, and Microsoft SQL Server. The application enables users to search available rooms, make and cancel reservations, simulate payments, and view booking details. It supports room categorization, automatic room allocation based on availability, reservation tracking, and booking validation through a structured service-based architecture.

## Features

* View all rooms
* View available rooms
* Room categorization:

  * Standard
  * Deluxe
  * Suite
* Automatic room assignment based on availability
* Create reservations
* Cancel reservations
* Payment simulation
* Transaction tracking
* Check-in and check-out date validation
* Reservation detail management
* Database persistence using SQL Server

## Technologies Used

* Java
* Spring Boot
* Spring Data JPA (Hibernate)
* Microsoft SQL Server
* Lombok
* Maven
* Object-Oriented Programming (OOP)
