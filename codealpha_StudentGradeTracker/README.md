# Task 1 - Student Grade Tracker

## Project Description

Student Grade Tracker is a console-based Java application developed to manage student academic records efficiently. The program allows users to add students, store subject marks, calculate performance statistics, and display reports through a terminal interface.

This project demonstrates the use of Object-Oriented Programming (OOP) concepts and Java collections such as ArrayList and HashMap.

---

## Features

* Add Student Records
* Validate Marks Input (0–100)
* Prevent Duplicate Roll Numbers
* View All Students
* Search Student by Name
* Calculate:

  * Average Marks
  * Highest Marks
  * Lowest Marks
* Show Class Topper
* Calculate Class Average
* Menu-driven Console Interface

---

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* HashMap
* Exception Handling

## How to Run

### Prerequisites

* JDK 17 or above installed
* Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or terminal

### Steps

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd StudentGradeTracker
```

3. Compile the project:

```bash
javac *.java
```

4. Run the application:

```bash
java Main
```

---

## How to Use

After launching the application, a menu will be displayed.

### Example Menu

```text
1. Add Student
2. View All Students
3. Search Student
4. Show Class Topper
5. Show Class Average
6. Exit
```

### Add Student

* Enter student name
* Enter roll number
* Enter marks
* System validates marks and prevents duplicate roll numbers

### View All Students

Displays all stored student records along with their marks and performance statistics.

### Search Student

Search for a student using their name and view complete details.

### Performance Statistics

The system automatically calculates:

* Average Marks
* Highest Marks
* Lowest Marks
* Class Average
* Class Topper

### Exit

Safely closes the application.
